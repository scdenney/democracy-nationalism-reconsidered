# Codebook

Analysis-ready datasets generated by `code/build_analysis_data.py`. All attitude measures are rescaled to [0,1], oriented so higher = more of the named concept (verified against the ISSP ZA10010 value labels).

## `outputs/analysis_individual.csv`

Individual level. N rows: 43,517 (29 countries).

| Variable | Type | Range | Description |
|----------|------|-------|-------------|
| `country_code` | int | ISO-3166 numeric | Country identifier |
| `country_name`, `region`, `region_regime` | str | — | Country and region/regime group |
| `c_sample` | int | — | ISSP sample id (DE, IL each split into two) |
| `WEIGHT_COM` | num | >0 | Combined ISSP weight |
| `regime_binary`, `vdem_row_2023`, `row_2023` | str | — | Regime classification (V-Dem RoW 2023) |
| `v2x_polyarchy`, `v2x_liberal`, `v2x_libdem`, `v2x_partipdem` | num | 0–1 | V-Dem v15 (2023) component indices |
| `democracy_rights_liberal` | num | 0–1 | Liberal democratic-rights index (minority rights, participation, civil disobedience) |
| `democracy_works_today` | num | 0–1 | Perceived democratic performance |
| `ascriptive_membership`, `civic_membership`, `national_superiority`, `protective_sovereignty`, `anti_immigrant_exclusion`, `national_pride_non_democracy`, `pride_democracy` | num | 0–1 | Nationalism dimensions |
| `age`, `female`, `education_level`, `left_right` | num | — | Individual controls (age years; female 0/1; ISCED 0–8; left–right 0–1) |
| `cm_ascriptive_membership`, `cm_civic_membership`, `cm_civic_minus_ethnic`, `cm_democracy_rights_liberal` | num | 0–1 | Country-mean moderators (level-2) |
| CFA item columns (e.g. `born_important`, `immigrants_increase_crime`, …) | num | 0–1 | Individual scaled items for the measurement models |

## `outputs/country_level.csv`

Country level. N rows: 29.

| Variable | Type | Description |
|----------|------|-------------|
| `country_code`, `country_name`, `region_regime`, `regime_binary`, `row_2023` | — | Country identifiers and classification |
| `v2x_polyarchy`, `v2x_liberal`, `v2x_libdem`, `v2x_partipdem` | num | V-Dem v15 (2023) |
| `cm_<measure>` | num | Country weighted means of each individual measure |
| `cm_civic_minus_ethnic` | num | Civic minus ascriptive membership (civic–ethnic balance) |
| `n` | int | Respondents |
